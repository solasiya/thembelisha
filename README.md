# Thembelisha Day Care & Pre-School Website

This is the official website for Thembelisha Day Care & Pre-School, designed to provide information about our programs, team, and legacy.

## Features
- **Responsive Design**: Looks great on mobile, tablet, and desktop.
- **Program Details**: Information for Babies, RR Class, and Grade R.
- **Gallery**: Interactive photo slideshow.
- **Contact Form**: Powered by EmailJS for direct inquiries.
- **Newsletter**: Stay updated with our latest news.

## Setup & Deployment
- [EmailJS Setup Guide](EMAILJS_SETUP.md) - How to activate the contact and newsletter forms.
- [Render Deployment Guide](RENDER_DEPLOYMENT.md) - How to host your website for free on Render.

## Local Development
To run this project locally:
1. Ensure you have Node.js installed.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the local server.
